#### Math Operations
**Addition**: 50 + 5<br>
**Subtraction** : 50 - 1<br>
**Multiplication** : 50 * 7<br>
**Division** : 400 / 25<br>
**Modulo** : 27 % 2<br>
**제곱** : 3 ** 2

**PEMDAS**: 괄호, 지수, 곱하기, 나누기, 더하기, 빼기

**NaN** : 숫자타입이지만 숫자가 아님을 표현하는 것 예) 0/0
